# Turkish Crisis Sentiment and Urgency Dataset

This repository provides access information for the manually labeled dataset used in the study:

**“Multi-Task Transformer Architectures for Simultaneous Sentiment and Urgency Classification of Turkish Crisis Communications.”**

The dataset consists of Turkish earthquake-related tweets that were manually annotated for:

- **Sentiment**: Negative, Neutral
- **Urgency**: U0, U1, U2, U3, U4

## Dataset Access

The labeled dataset is publicly available on Kaggle at:

**https://www.kaggle.com/datasets/abdo94ma/scoreddataset**

## Dataset Description

This dataset was prepared for research on Turkish crisis communication following the February 6, 2023 Kahramanmaraş earthquakes. It includes manually labeled tweet data designed for:

- binary sentiment classification
- five-level urgency classification
- multi-task learning experiments
- comparative evaluation of classical, deep learning, and transformer-based models

## Labels

### Sentiment Labels
- **Negative**
- **Neutral**

### Urgency Labels
- **U0**: Informational / non-urgent
- **U1**: Low urgency
- **U2**: Moderate urgency
- **U3**: High urgency
- **U4**: Extreme urgency
